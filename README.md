# CampusFlow

CampusFlow is an AI-powered student productivity platform that helps students manage deadlines, assignments, exam schedules, and college notices.

## Features

* Student Authentication
* Assignment Tracking
* AI Notice Summarization
* WhatsApp Notifications
* Google Calendar Integration
* n8n Workflow Automation

## Team Structure

* Frontend: frontend-dev
* Backend: backend-dev
* Automation: automation-dev
* AI: ai-dev
